Returns list of subopt structures or writes to fp. 

This function produces {\bfseries all} suboptimal secondary structures within 'delta' $\ast$ 0.\-01 kcal/mol of the optimum, see \cite{wuchty:1999}. The results are either directly written to a 'fp' (if 'fp' is not N\-U\-L\-L), or (fp==N\-U\-L\-L) returned in a \#vrna\-\_\-subopt\-\_\-solution\-\_\-t $\ast$ list terminated by an entry were the 'structure' member is N\-U\-L\-L.
